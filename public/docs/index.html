Introduction

Base URL: http://localhost:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Register

Регистрация нового пользователя

Example request:
curl --request POST \
    "http://localhost:8000/api/register" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "name=dpowwzdppamaxvjrwilv"\
    --form "email=filiberto.mclaughlin@example.com"\
    --form "password={&:?2OjAcJ"\
    --form "password_confirmation=perferendis"\
    --form "avatar=@/tmp/phpOSRCKt" 
const url = new URL(
    "http://localhost:8000/api/register"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('name', 'dpowwzdppamaxvjrwilv');
body.append('email', 'filiberto.mclaughlin@example.com');
body.append('password', '{&:?2OjAcJ');
body.append('password_confirmation', 'perferendis');
body.append('avatar', document.querySelector('input[name="avatar"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
