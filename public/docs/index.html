Introduction

Base URL: http://localhost:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Auth

Авторизация пользователей и всё что с ней связано.

Login

Авторизация пользователя

Example request:
curl --request POST \
    "http://localhost:8000/api/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"lessie.beer@example.org\",
    \"password\": \"8B]CSP=BDMj;6\\\\j)|y\"
}"

const url = new URL(
    "http://localhost:8000/api/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "lessie.beer@example.org",
    "password": "8B]CSP=BDMj;6\\j)|y"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):


{
    "status": "success",
    "message": "messages.auth.login.success",
    "access_token": "3|OnfjrdKtmnz2ysgeEWJNTCExlBq3eV0Ga7nRxCR54b7ce52c",
    "token_type": "Bearer",
    "user": {
        "name": "Фёдоров Святослав Владимирович",
        "email": "veselov.leonid@example.com",
        "roles": []
    }
}

Example response (403):


{
    "status": "success",
    "message": "messages.auth.login.error"
}

create

requires authentication

Метод создания трека

Example request:
curl --request POST \
    "http://localhost:8000/api/tracks" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "name=dspyprmwoveqxmuphewznsngpdhjwattpioduw"\
    --form "address=at"\
    --form "latitude=-87"\
    --form "longitude=-180"\
    --form "images[]=@/tmp/phprzBfbT" 
const url = new URL(
    "http://localhost:8000/api/tracks"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('name', 'dspyprmwoveqxmuphewznsngpdhjwattpioduw');
body.append('address', 'at');
body.append('latitude', '-87');
body.append('longitude', '-180');
body.append('images[]', document.querySelector('input[name="images[]"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());
